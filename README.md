For Task A, if you want to localize this deployment, I have provided the Ubuntu tutorial here.

Install npm
 
~~~
sudo apt install npm
~~~

Install node
 
~~~
npm install node
~~~

Install missing dependence

~~~
npm install
~~~

Run demo

~~~
node app.js
~~~

Open your browser and jump to: http://localhost:5000
